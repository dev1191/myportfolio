# 🛠️ DEV RAJ Thapa

A high-performance, developer-centric portfolio built with **Vue 3**, **Vite**, and **Tailwind CSS 4**. This project features a unique "Stitched" design aesthetic, high-speed animations, and a focus on technical precision.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🏗️ Technical Architecture

- **Framework**: [Vue 3](https://vuejs.org/) (Composition API)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) with a custom design system
- **Animations**: [@vueuse/motion](https://motion.vueuse.org/) for fluid entrance transitions
- **Icons**: [Lucide Vue Next](https://lucide.dev/) for consistent, type-safe iconography
- **State**: Reactive Composables for theme management and UI state

## 🎨 Design Philosophy: "STITCHED"

The UI is inspired by high-end technical interfaces, utilizing a modular grid system where components are "stitched" together by fine-line borders.

- **Grid System**: 1px gap patterns mimicking professional CAD/Blueprint software.
- **Color Palette**: Electric Green (#75ff9e) primary on JetBlack (#131313) surfaces.
- **Typography**: [JetBrains Mono](https://www.jetbrains.com/lp/mono/) for technical data and [Geist](https://vercel.com/font) for clean reading.

## 📁 Project Structure

```text
src/
├── components/     # Atomic UI components (Hero, TechStack, ProjectCard, etc.)
├── composables/    # Reusable logic (useTheme, useNavigation)
├── assets/         # Global styles and branding assets
├── App.vue         # Main entry point and layout
└── style.css       # Tailwind 4 configuration and global design tokens
```
